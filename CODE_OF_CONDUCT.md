# Code of Conduct

## Our pledge

This project welcomes contributions and participation from anyone, regardless of background, identity, or experience level.

## Expected behavior

- Be respectful and constructive in discussions
- Assume good intent; ask clarifying questions before reacting
- Give credit where credit is due
- Help others learn

## Unacceptable behavior

- Harassment, insults, or personal attacks
- Discriminatory language or imagery
- Publishing others' private information
- Spam, off-topic promotion, or trolling
- Anything that makes the project a worse place to be

## Enforcement

Issues can be reported privately by emailing the maintainer or opening a private security advisory. The maintainer will review reports and take appropriate action (warning, temporary ban, or permanent ban).

## Scope

This Code of Conduct applies in all project spaces — issues, pull requests, discussions, and any public communication that mentions or represents the project.
